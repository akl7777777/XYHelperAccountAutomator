# XYHelperAccountAutomator
xyhelper免费版批量添加账号的脚本

这个项目是一个用于批量添加账号到xyhelper的脚本。它首先定义了一个包含多个账号信息的字符串，然后通过拆分这个字符串来获取每个账号的信息。然后，它使用一个循环来处理每个账号，通过模拟用户的操作（点击按钮，输入信息）来添加账号。在处理每个账号之间，它会暂停一段时间（默认为5000毫秒）。
